01   Problem

Everything ran through a person's inbox

A mid-size office was fielding 330+ facility requests every month - AC repairs, IT issues, courier pickups, parking slots. Each one required an admin to manually type it into a tracking sheet, send a confirmation email to the employee, and CC the department head. That is 5 minutes of repetitive data entry per request, across 22 working days.

  • 27.5 admin hours burned every month on copy-paste work
  • Employees waited hours - sometimes days - for any acknowledgment
  • Requests buried in email inboxes, no ticket numbers, no audit trail
  • 5-10% of requests quietly fell through the cracks
02   Solution

A pipeline that runs itself

Built an end-to-end automated ticketing system using n8n as the workflow engine. An employee fills out a web form in 60 seconds. The moment they submit, four things happen automatically - with zero manual work.

01
Web Form
Employee selects a request type (AC, IT support, courier, parking, etc.), fills in their name, email, department, date needed, and any extra details. Takes about 60 seconds. No phone calls, no emails.
02
Webhook Trigger
The moment the employee hits submit, the form data is sent via HTTP POST to n8n's webhook URL. n8n wakes up instantly and begins processing - no polling, no delay, no human checking an inbox.
03
Unique Ticket ID
n8n generates a timestamp-based reference number (e.g. TKT-1714892341000) unique to that submission. This ID is attached to every downstream action - the spreadsheet row, both emails - so nothing gets mixed up or lost.
04
Google Sheets Log
All form fields plus the ticket ID and timestamp are appended as a new row in a shared Google Sheet. Every request is now searchable, sortable, and visible to anyone with access - a permanent audit trail replacing the old email chaos.
05
Dual Email Alerts
Two emails fire simultaneously. The employee receives a confirmation showing their ticket number - proof the request is logged. The admin receives the full request details ready to act on. Both arrive in under 60 seconds from form submission.
